# 📂 Laboratoire de Simulation RD FORMATION 🎓

Bienvenue dans ce dépôt dédié à la conception de **sites web pédagogiques fictifs**. Ces outils sont développés pour servir de supports d'apprentissage immersifs dans le cadre de formations professionnelles (Tertiaire, Commerce, Gestion).

---

### 🎯 Objectif du Projet
L'objectif est de pallier l'absence de logiciels métiers réels en formation en proposant des **environnements simulés** sécurisés. Ces sites permettent aux apprenants de manipuler des interfaces réalistes, de pratiquer des gestes métiers et de valider des compétences techniques sans risque d'erreur sur des données réelles.

### 🚀 Exemples de Scénarios Pédagogiques
Chaque site ici présent est conçu autour d'un scénario spécifique :

1.  **[L'Échoppe Gauloise](https://rodulfo-dmgz.github.io/rd-gestion/lechoppe-gauloise/)** : 
    * **Public :** Métiers du tertiaire (Bac Pro, BTS).
    * **Usage :** Simulation de création d'entreprise, gestion des stocks, facturation et prise de commande.
    * **Compétences visées :** Administration des ventes, suivi comptable, gestion de base de données clients.

### 🛠️ Architecture Technique
Pour garantir une accessibilité maximale en salle de formation, ces sites utilisent une stack légère et robuste :
* **Front-end :** HTML5 / CSS3 (Responsive Design pour tablettes/PC).
* **Logique :** JavaScript (Calculs de marges, TVA, gestion dynamique de panier).
* **Interactivité :** Intégration possible avec **H5P** et **Moodle** via iFrame pour le suivi des scores (SCORM/xAPI).

---

### ⚙️ Installation & Utilisation
Ces sites sont conçus pour être déployés via **GitHub Pages**. 
1. Clonez le dépôt.
2. Ouvrez `index.html` dans un navigateur ou accédez aux liens de démonstration ci-dessus.
3. Intégrez-les directement dans votre LMS (Moodle, Google Classroom) pour enrichir vos parcours.
